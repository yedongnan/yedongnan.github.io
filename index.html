成功公式：好的团队什么样

发表于 2019-07-01 | | 阅读次数

　　现代意义上的成功已经很少能够仅靠个人独立完成，一个项目的完成需要的是一个团队的通力协作，一个好的公司的战斗力不应体现于某个领导的一言堂，我们需要大家聚在一起，分工合作，各自发挥各自的作用。
　　丰田公司的生产线可能和世界上其他大公司的生产线都有所不同。类似宝马奔驰等公司都是不允许生产线的工人随意叫停，每次叫停，生产线的效率都会大受影响，而丰田的生产线却可以由工人自己决定，只要他觉得哪里出了问题，随时喊停，马上就会有辅助小组的人过来询问怎么了，并商讨解决办法。丰田的生产线每天都会被叫停1000次之多，这要是放在其他汽车公司，都是不可想象的。而且，丰田的这些经验，其他厂商想学也学不走，它们存在于每个工人的脑子里，拼接起来构成了丰田的制造优势。
　　招一群单兵能力出众的成员组成一个团队其战斗力就一定好吗？确实，根据木桶原理，组成团队的每一个环节都不差，这个团队没有明显的短板，战斗力应该不差了吧，结果是未必如此。每年的NBA赛季结束之后都会搞一个全明星赛，每个队伍的阵容都超级豪华，全明星赛大家都很卖力的表演，但这样的队伍未必适合打正赛，这样的队伍强则强矣，但大家都是大牌，不会长久的相安无事，内讧迟早会一点一点的消耗掉整个队伍的凝聚力。
　　达尔文提出过群体演化的思想，认为动物的演化并不仅仅是个体跟个体竞争，还有群体跟群体的竞争。一个团队的战斗力靠的是每个人的付出，如果一个人个性太突出，锋芒毕露，即使能力再强，反而不利于整个团队的协作与输出。与那些齐心协力的群体相比，竞争力都在内耗中失散掉了。
　　强人 + 强人 ≠ 强大团队。群体的演化，需要各种类型的人好好配合才行。
　　巴拉巴西援引了别人对百老汇史上全部474个音乐剧和爵士乐史上全部10万首曲子的研究，想分析出哪些因素可以造就团队的成功。这里的团队成功标准是演出有没有大获成功，获得观众的喜爱。而喜爱 = 熟悉 + 意外。即既有继承，又有创新。研究的结论是要想让团队成功，各成员之间既不能全是强联系，也不能全是弱联系。强联系带来熟门熟路，不至于太出离观众的眼界，弱联系则碰撞出火花，给观众带来惊喜。这样，有熟人，有新人，大家坐在一起，平等的交流，这样才有可能让大家的想法充分表达和碰撞。
　　想法多了，团队必须有一个做决定的人，研究发现，通常来说，一个项目中最重要的工作，几乎都是其中的一个人做的。这个人是这个项目的主导，他按他心中的所思所想，一步一步的将这个项目往前推进。这就是巴拉巴西所说的，团队可以有很多人，但整个项目定于一人，其他人可能都是打辅助的，打打边鼓，起点出谋划策的工作。
　　巴拉巴西给出了成功团队的网络结构，大约有三个特点。
　　第一，定于一人。这个人为项目负总责，其他人提供支持。
　　第二，人员构成要多样化。不能都是强联系，也不能都是弱联系
　　第三，团队成员坐在一起，平等的进行交流。

成功公式：欢迎度与适应度

发表于 2019-07-01 | | 阅读次数

　　成功的作家出的作品比普通作家更易出版和畅销，成功的艺人拍的片子更易获得大众的关注，普通人挣点钱要把汗珠摔成八瓣，而王健林定的小目标是先挣他一个亿。过往的成功似乎更容易带来下一个成功，我们跑到电影院看电影，也要看看那部电影是熟悉的导演拍的或者知名演员的作品，如果都不是，进影厅的心情都是忐忑的，这么看来，“马太效应”时刻都在影响着我们。那么这里要问的问题是，到底是名人的光环加持导致名人前一个的成功带来的下一个成功，还是名人的言行和做事方式本身就给其带来成功呢？
　　J.K.罗琳的例子很能说明问题，她写的《哈利波特》大获成功。2013年她做了一个实验，看能否单纯靠实力再让小说畅销一次，她以匿名的方式写了两本小说，结果是一开始还被出版社拒稿，即使出版后销量也很不理想，直到引起了一些人的猜测，J.K.罗琳承认了是她的作品后，两本小说的销量暴增。
　　而研究表明，类似流量明星这种，花钱雇托儿带来的成功甚至也可以带来成功，这里面的机制是“可被奖励性”，我们在给某个人点赞和奖励的时候，总会参考之前有没有人奖励过，如果有，则自己的奖励就是安全的。我们虽然不喜欢微博上那些转发评论动不动就上千万的流量明星，但他们确实获得了足够的知名度。
　　从这个角度说，人真不能做坏事，你留下的好印象确实是会叠加的。做事靠谱，让人心悦诚服，时间久了，你获得的“奖励”或“点赞”就会越来越多，也就更容易获得别人的“奖励”和“点赞”，个人的声望莫不由此而來。
　　下面要讲的是适应度。旧的成功总在老去，总有新东西冒出来。一家优秀的企业成立之初也许会磕磕碰碰，这时候知名度是不够的，但他们却能在竞争中不断壮大。阿里巴巴成立之初在与国外电商巨头的竞争中崭露头角，展现出独特的竞争优势，获取先发地位，这种成功带来了之后十多年的持续成功。同样的，欲取而代之的互联网企业比比皆是，前有京东，后有拼多多，这些企业都在残酷的市场竞争中杀出了一片天，确实满足了特定细分市场的需求，这种面对竞争时的能力被称为适应度。
　　巴拉巴西由此给出了《成功公式》关于成败的第三条定律：未来的成功 = 欢迎度 × 适应度。
　　欢迎度定义了它之前的成功，适应度反映的则是它在面临竞争时优势如何。欢迎度可以通过不同方式获得，可以通过积累，炒作也行，适应度决定了你能够走多远。
